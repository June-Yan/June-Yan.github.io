Vibes

  1. 看了这个视频，讲的是婚姻的本质、一些年轻人的困境。非常感到感同身受，即便我认为我比博主的条件可能还会好一点。我个人认为婚姻是爱情的结果，不是目的。我并不是否定不奔着结婚的恋爱是耍流氓这个观点，但至少要找和自己三观一致，知识、技能、能力、财富等差不多的女生并在婚前要有一段时间的必要交往甚至同居，我排斥相亲、排斥没有共同生活经历的人突然之间就要生活在一起。同居并不是做爱，是至少把生活习性和柴米油盐先预演一遍，如果婚前都不能接受的话、尽早收手不要祸害小孩是更好的选择。 如果结婚只是为了花钱充面子、那么不应当在比较年轻的年纪结婚（虽然这句话的前提是你要有足够的上升前景，导致这个年纪提升自己效益更大；但即便你年轻但没有上升空间，也不应当在自己的生活中加入一项负债）；如果结婚之后要生小孩，那么同样不应该尽早结婚，因为你根本没有足够积蓄去托举小孩，这只是让孩子再吃一遍你的苦罢了，阶级固化的大背景下只是一代一代人的循环、自我提升和阶级跃迁都极其困难，金钱学历的通货膨胀，只是营造一个错觉，你的处境、地位、消费力和上一辈没有本质区别。 个人而言，在经济水平有限的情况下，投资一段新的感情或者新的生命，远不如投资或者犒劳自己实用。人人都难成为孙宇晨，杜绝房车投资自己且大赚特赚；但是能成为更好的自己、褪去二十多年的疲惫就已经不错了。至于社会舆论和父母亲戚，别听、也不要纠缠，做好课题分离，孩子只负责他们的健康，如何过完这一生、找寻自己的价值，都是孩子的课题。
  2. cs336的前提是DL和RL的了解，吴恩达+cs231n+cs224n这样的基础会很好入门。 It’s a good project for复试。
  3. 还是需要周期性的计划，类似甘特图那种，明确某个项目指定的完成时间段。
  4. AI牛逼。obsidian的美化和数据可视化（dataviewjs/css/mermaid）完全可以依赖AI，减少了很多学习成本。
  5. 刷到一个复旦大三数科的，交易策略是主观+量化。emmmm感觉挺厉害的，而且数学专业能自学cs和深度学习，把图论运用到交易，自学能力和自驱力都好强。等身体恢复、课程减少之后试一试。
  6. 不知道会不会挂科，主要是计算机网络。我承认我大意了没有闪。老妈问要不要求求情，but in fact I probably can’t pass even if I got a full marks on regular assignment.

Daily Review

  5. 娱乐方式和质量需要提高
